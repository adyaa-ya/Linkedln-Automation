# LinkedIn Automation using Python

## Objective
The objective of this project is to automate basic LinkedIn tasks such as:
- Logging into LinkedIn
- Searching people using keywords
- Sending connection requests automatically

## Tools & Technologies
- Python
- Selenium WebDriver
- Chrome Browser
- WebDriver Manager
